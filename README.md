# 简历评分工具 - 应届生专版

一个专为应届毕业生设计的智能简历评分系统，提供专业的简历分析、岗位推荐和改进建议。

## 🌟 项目特色

### 🎯 专为应届生定制
- **学历权重优化**：采用学历学校加权评分体系
- **校园经历识别**：专门识别学术竞赛、奖学金、社团活动等校园经历
- **技能专精检测**：自动识别编程、设计、数据分析等专业领域的技能专精度

### 🚀 智能分析引擎
- **多格式支持**：支持PDF、Word文档上传和文本直接粘贴
- **全面评分体系**：覆盖基本信息、教育背景、专业技能、实践经验、奖励荣誉五大维度
- **专精加成系统**：技能专精者可获得额外加分，突破100分上限

### 💡 个性化建议
- **精准岗位推荐**：基于技能匹配度推荐最适合的职业方向
- **针对性改进建议**：提供具体可行的简历优化方案
- **学术导向识别**：为有学术倾向的学生推荐研究型岗位

## 🛠️ 技术架构

### 前端技术栈
- **原生JavaScript**：轻量级实现，无框架依赖
- **现代CSS**：响应式设计，支持移动端适配
- **PDF.js**：客户端PDF文档解析
- **Mammoth.js**：Word文档内容提取

### 核心算法
- **智能文本解析**：多模式匹配识别简历关键信息
- **分级评分系统**：基于行业标准的量化评分模型
- **专精检测算法**：深度分析技能分布，识别专业特长

## 📊 评分体系详解

### 基础评分（满分100分）
- **基本信息（10分）**：姓名、电话、邮箱、地址
- **教育背景（30分）**：学校层次15分 + 学术表现8分 + 专业匹配7分
- **专业技能（25分）**：编程8分 + 设计5分 + 数据分析5分 + 其他7分
- **实践经验（25分）**：实习经历15分 + 项目经验7分 + 经验质量3分
- **奖励荣誉（10分）**：奖学金3分 + 竞赛4分 + 证书2分 + 领导力1分

### 专精加成系统
- **编程专精**：掌握5+技术可获得最高8分加成
- **学术专精**：3+学术成果可获得5分加成
- **设计专精**：3+设计技能可获得6分加成
- **实践专精**：丰富项目经验可获得4分加成

### 上传简历
1. **文件上传**：支持拖拽或点击上传PDF/Word文档
2. **文本粘贴**：直接粘贴简历文本内容
3. **格式要求**：文件大小不超过10MB

- **总分显示**：展示基础分数和专精加成
- **详细评分**：各维度具体得分和改进空间
- **岗位推荐**：基于技能匹配的职业建议
- **优化建议**：针对性的简历改进方案
